Privacy Policy for Claire the Explain Bear

Last Updated: March 05, 2025

Claire the Explain Bear ("the Extension") is a Chrome extension developed to assist users in understanding text on webpages by providing detailed explanations powered by OpenAI's API. This privacy policy explains how we handle your data when you use the Extension.

1. Data We Collect

We collect the following data to provide the Extension's functionality:

  • Highlighted Text: When you highlight text on a webpage and click "Explain," the selected text is sent to OpenAI's API to generate an explanation.
  • Page Content: Depending on your context settings (configurable in the options page), portions of the webpage content—or the entire page if the "Full Page" option is selected—may be sent to OpenAI to provide context for the explanation.
  • Additional Prompt: If you provide an additional prompt in the "Explain Further" feature, this text is also sent to OpenAI to refine the explanation.
  • OpenAI API Key: You provide your OpenAI API key in the Extension's options page, which is stored locally in Chrome storage (chrome.storage.sync) and used to authenticate requests to OpenAI. This key is not shared with any other parties.

2. How We Use Your Data

The data collected is used solely to provide the Extension’s core functionality:

  • The highlighted text, page content, and additional prompts are sent to OpenAI's API to generate explanations, which are then displayed in a popup within the Extension.
  • The OpenAI API key is used to authenticate API requests to OpenAI on your behalf.

We do not store or log any of this data on our servers. All processing happens between your browser and OpenAI's API.

3. Third-Party Services

The Extension relies on the following third-party services:

  • OpenAI: The Extension uses OpenAI's API to generate explanations. Your highlighted text, page content, and additional prompts are sent to OpenAI for processing. OpenAI may retain this data as per their policies. Please review OpenAI's privacy policy at https://openai.com/privacy for details on how they handle your data.
  • Flaticon: The bear icon used in the Extension is provided by Freepik via Flaticon. No user data is shared with Flaticon. The icon is credited in the Extension’s options page and Chrome Web Store listing.

4. Data Storage

The Extension stores the following data locally on your device:

  • OpenAI API Key: Stored in Chrome storage (chrome.storage.sync) and encrypted by Chrome. This key is only accessed to make API requests to OpenAI.
  • Context Settings: Your preferences for context settings (e.g., nearest paragraph, full page) are stored in Chrome storage to persist across sessions.

No other user data (e.g., highlighted text, page content) is stored by the Extension. All such data is processed in real-time and discarded after the API request is complete.

5. Data Sharing

We do not share your data with any parties other than OpenAI, as described in Section 3. The Extension does not transmit your data to our servers or any other third parties beyond what is necessary for its functionality.

6. Your Choices

You have control over how the Extension handles your data:

  • Context Settings: You can configure how much webpage content is sent to OpenAI by adjusting the context settings in the Extension's options page. For example, selecting "Full Page" sends the entire page, while other options limit the context to smaller portions.
  • API Key: You can remove your OpenAI API key from the options page at any time, which will prevent further API requests until a new key is provided.
  • Usage: You can stop using the Extension at any time by uninstalling it from Chrome, which will remove all stored data (API key, settings) from Chrome storage.

7. Security

We take the following measures to protect your data:

  • The OpenAI API key is stored in Chrome storage, which is encrypted by Chrome to prevent unauthorized access.
  • User inputs (highlighted text, additional prompts) are sanitized to prevent injection attacks before being sent to OpenAI.
  • No user data is logged or stored on our servers.

However, since the API key is stored locally, you should ensure your device is secure to prevent unauthorized access to your browser data.

8. Changes to This Policy

We may update this privacy policy from time to time to reflect changes in the Extension’s functionality or legal requirements. Any updates will be posted on this page, and the "Last Updated" date will be revised accordingly. We encourage you to review this policy periodically.

9. Contact Us

If you have any questions or concerns about this privacy policy or the Extension’s data practices, please contact us at your-email@example.com.
